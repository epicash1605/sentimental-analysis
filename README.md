# Sentiment Analysis

NLP project that analyzes text sentiment and summarizes results with simple visuals.

## Project Summary
This project processes text data, applies sentiment scoring, and visualizes trends across samples.

## Key Features
- Text preprocessing
- Sentiment scoring
- Basic aggregation and visualization

## Tools & Libraries
- Python
- pandas
- numpy
- matplotlib

## How to Run
```bash
pip install -r requirements.txt
python main.py
